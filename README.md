# Psychology Management System Frontend

Angular frontend for the Psychology Management System.

This application provides the user interface for managing patients, clinical records, appointments and documents while consuming the backend REST API.

---

# Purpose

This repository contains the frontend implementation only.

Detailed technical documentation is intentionally separated into dedicated documents under `/docs`.

---

# Technology Stack

* Angular 21
* TypeScript
* SCSS
* Angular Signals
* CoreUI
* Bootstrap
* Angular Material (where appropriate)
