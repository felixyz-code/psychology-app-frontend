# Project

> Product vision for the Psychology Management System Frontend.

---

# Vision

The frontend provides the complete user experience for the Psychology Management System.

Its purpose is to deliver a modern, responsive and maintainable application while consuming the backend API without duplicating business logic.

---

# MVP Scope

The current MVP includes:

* Authentication
* Dashboard
* Patients
* Case Files
* Session Notes
* Documents
* Appointments

The MVP targets independent psychologists.

---

# Product Principles

The frontend follows these principles:

* Backend First
* Feature-Based Architecture
* Reusable Components
* Responsive Design
* Consistent User Experience
* Simple Clinical Workflows
