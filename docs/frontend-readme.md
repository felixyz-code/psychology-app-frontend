# Frontend Readme

## Vision general del proyecto

Este frontend corresponde a una aplicacion clinica de psicologia y neurodesarrollo construida con Angular 21. Su objetivo es ofrecer una base mantenible, escalable y clara para consumir el backend existente sin duplicar logica de negocio ni asumir comportamientos fuera de la API disponible.

El proyecto se desarrollara con Angular Material, componentes standalone, Angular Signals y estilos en SCSS. La integracion con backend seguira un enfoque backend first, donde NestJS actua como fuente de verdad para autenticacion, datos clinicos y reglas operativas.
